# Mytrip - Travel & Tour Management System

A premium PHP-based travel management platform for booking packages, managing trips, and exploring destinations.

## Features
- **User Authentication**: Secure login, registration, and password reset with OTP verification.
- **Package Management**: Browse and search for travel packages with detailed information and images.
- **Booking System**: Seamless checkout process integrated with Razorpay.
- **Admin Dashboard**: Full control over packages, users, bookings, and reviews.
- **Budget Tool**: Plan your trips effectively with our built-in budget estimator.
- **Favorites & Support**: Save your favorite destinations and get 24/7 support.

## Tech Stack
- **Backend**: PHP
- **Database**: MySQL
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Payment Gateway**: Razorpay
- **Design**: Premium, responsive UI with modern aesthetics.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/jayrajchabhad/Mytrip.git
   ```
2. Import the database from `Database/mytrip.sql` into your MySQL server.
3. Update `db.php` with your database credentials.
4. Update `config_razorpay.php` with your Razorpay API keys.
5. Access the project via your local server (e.g., XAMPP).
